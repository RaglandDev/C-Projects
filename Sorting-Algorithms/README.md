# Sorting Algorithms

## Build & Run
```
$ make all
$ ./sorting [options] 
```
## Clean Directory
```
$ make clean
$ make cleangraph
```
## Generate Graphs
```
$ chmod +x plot.sh
$ ./plot.sh
```
